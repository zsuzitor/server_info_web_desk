подробнее о блоке Info
все делится на статьи и секции
статья- аналог текстового файла (1 поле для названия 2 поле для содержимого и хештегов)
секция-аналог папки(1 поле для названия) может содержать другие секции, статьи и находиться в некой секции

4 основных блока:
1 сверху блок для управления (стрелки влево\вправо изменяют размер окон,
 линия пер середине тоже изменяет размер окон, ее надо перетаскивать,
стрелка вниз\вверх отображает\убирает блок №2

2блок с кнопками, самый левый, выезжает после нажатия на стрелку "вниз" на блоке №1
(седержит основные кнопки,при наведении появляется название)

3блок с содержимым данных(названия секций , статей и их иерархия)

4 блок отображения\редактирования и для некоторых других функций для которых нужно наприме поле для ввода

также блок для поиска(находится верху, при наведении расширяется и поиск по кнопке)
